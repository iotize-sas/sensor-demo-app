# Development
### Installation

```
git clone https://github.com/iotize-sas/sensor-demo-app.git 
cd sensor-demo-app
npm install
```

## Run application

### On a web server

```shell
npm run webapp
```

### On Android

```shell
npm run android -- --prod
```
